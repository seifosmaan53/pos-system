# 🏪 POS System - Complete Business Solution

## 🚀 Quick Start (2 minutes)

### For Windows:
1. **Double-click:** `INSTALL-AND-START.bat`
2. **Wait for installation** (first time only)
3. **System opens automatically** in your browser

### For Mac/Linux:
1. **Open Terminal** in this folder
2. **Run:** `chmod +x *.sh && ./install-and-start.sh`
3. **System opens automatically** in your browser

## 🎯 How to Use

### First Time Setup:
1. **Connect flash drive** to register computer
2. **Copy all files** to computer's hard drive
3. **Run installation** (one of the files above)
4. **System installs** Node.js dependencies
5. **System starts** automatically

### Daily Use:
1. **Double-click** `START-POS.bat` (Windows) or `./start-pos.sh` (Mac/Linux)
2. **System opens** in browser at http://localhost:8080
3. **Start selling!** Add products, process sales, print receipts

### Backup Your Data:
- **Automatic:** System creates daily backups
- **Manual:** Use Backup Management page
- **Location:** `backups/` folder
